# Privacy Policy for Nmap Web Interface

Last updated: May 19, 2026

Nmap Web Interface ("we", "our", or "the Extension") is committed to protecting your privacy. This Privacy Policy explains how our Chrome Extension handles user data.

---

## 1. Information Gathering and Usage
Nmap Web Interface is a developer network auditing utility. 
- **Active Tab Data:** The Extension utilizes the `activeTab` permission strictly to read the domain name or IP address of the active website when you manually click the Extension icon to prepopulate your scanning fields. 
- **No Data Collection:** The Extension **does not collect, store, or transmit** any personally identifiable information (PII), browsing history, locations, or user credentials.
- **Local Processing:** All network audits and scan requests are initiated directly by the user. The commands are processed locally and queried against your own local API server to communicate with your host's Nmap binary. We do not run any remote tracking servers, database backups, or user analytics.

---

## 2. Data Transference to Third Parties
We do not sell, trade, or transfer any data processed by the Extension. The data (domain name/IP/command options) is strictly sent to your local machine's local server (localhost) to conduct network scans that you have authorized.

---

## 3. Policy Compliance
The Extension complies with the Google Chrome Web Store Developer Program Policies, including the User Data Policy's requirements for Limit Use.
